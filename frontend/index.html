🕸️ Análisis de Redes y Grafos (I+D)

Me hubiera encantado implementar un Modelo de Grafos (GNN) para este proyecto. Las redes son increíbles para detectar el fraude porque analizan las "relaciones" entre cuentas (quién transfiere a quién) en lugar de ver cada transacción de forma aislada, lo que es vital para detectar el lavado de dinero en cadena.

Sin embargo, llevar un modelo de grafos a producción requiere infraestructura pesada (como bases de datos Neo4j) y la interacción en la web se volvía demasiado compleja para una demo rápida. Aún así, aquí te dejo una visualización interactiva 3D que generé con Python mostrando cómo se ve una red de fraude en PaySim.

💡 Explora el universo 3D: Arrastra el ratón para rotar la cámara, haz scroll para hacer zoom y mantén presionado un nodo para moverlo.
